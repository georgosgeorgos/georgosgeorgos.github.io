# georgosgeorgos.github.io

Personal academic website for Giorgio Giannone.

## Structure

- `index.html` — Main single-page site (About, Research, Publications, Projects, Open-source, Datasets, Teaching, Patents, Theses)
- `css/style.css` — Stylesheet
- `blog/` — Jekyll-based blog
- `assets/` — Images, logos, CV
- `writing/` — Writing collection (essays, notes, predictions)

## Tech Stack

- Static HTML + CSS (no build system)
- Jekyll for the blog
- GitHub Pages for hosting
- Font: Ubuntu Mono

## Local Development

Open `index.html` in a browser. For the blog, run:

```sh
cd blog
bundle exec jekyll serve
```
